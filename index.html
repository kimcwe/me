kimchiGrafx

A passion project of mine, is my small YouTube channel where I integrate Data Analytics and Visualization into NBA statistics using mainly R, to produce a brief video form to explain or introduce certain trends or opinions about the NBA and its players!

RedSox Game Reporter

Simple Web Scraping script, written in python, to let my fellow Bostonians know if there is a Red Sox game that day! It will run in your respective shell, letting you know if there is a Red Sox game on that particular day, whether it is home or away, the date/time of the game, and an option to open up the full schedule in your browser.

FastQ Analyzer

A python program that opens and reads a given .fastq file, and determing various statistics for the respective file, for aiding in further downstream analysis. These include: length of seq, average quality score, GC%, proportions of good, average, and bad q-scores, etc.

FASTA Reader

A python program that reads the input .fasta file, and in turn provides an outfile that classifies whether the given input sequences are amino acids, RNA, or DNA sequences, and additionally gives its equivalent versions in the aforementioned formats, performing translation and transcription as necessary.
